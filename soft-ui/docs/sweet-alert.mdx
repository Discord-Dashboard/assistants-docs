---
title: "Sweet Alert"
description: "An Object which contains all the SweetAlert information."
---

```js
theme: SoftUI({
  sweetalert: {
    errors: {
      requirePremium: String,
    },
    success: {
      login: String,
    },
  },
});
```

## Example Usage

```js
theme: SoftUI({
  sweetalert: {
    errors: {
      requirePremium: "You need to be a premium member to do this.",
    },
    success: {
      login: "Successfully logged in.",
    },
  },
});
```
